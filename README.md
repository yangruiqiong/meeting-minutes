# 🎙️ 会议纪要助手

自动录音、转文字、生成会议纪要的桌面应用。

## 功能特点

- 🎙️ 一键录音（支持实时录制）
- 📂 支持导入已有音频文件（WAV/MP3/M4A等）
- 📝 本地 Whisper 模型转文字（离线，不联网，中文效果好）
- 📋 自动生成结构化会议纪要
- 💾 保存为 TXT 文件

## 使用方法

### 第一次运行（安装依赖）

双击 `run.bat`，会自动安装所需依赖包。

> ⚠️ 首次运行会下载 Whisper 语音模型（约 150MB），请确保网络畅通，耐心等待。

### 正常使用

1. 双击 `run.bat` 启动程序
2. 填写参会人员和会议主题（可选）
3. 点击 **▶ 开始录音** 开始录制
4. 会议结束后点击 **⏹ 停止录音**
5. 点击 **🔄 转文字并生成会议纪要**
6. 等待处理完成后，点击 **💾 保存会议纪要**

### 导入已有录音

点击 **📂 导入音频** 选择本地音频文件，再点击转写即可。

## 系统要求

- Windows 10/11
- Python 3.8+
- 麦克风（录音功能需要）

## 依赖包

- `sounddevice` - 录音
- `soundfile` - 音频文件读写
- `numpy` - 数据处理
- `openai-whisper` - 语音识别
